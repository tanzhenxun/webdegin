WEB DESIGN

BIBWB114 JAMES WOO

The module covers fundamental of web site development and the techniques of authoring Web pages with HTML and CSS. After the course, you should be able to :

  • Define the basic knowledge of Web site development.
  • Build web pages using HTML and CSS.
  • Produces a complete website using HTML, CSS and responsive web design.
